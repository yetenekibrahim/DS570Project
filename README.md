# DS570Project
Project Overview
This repository contains the end-to-end data science project developed by Ibrahim Yetenek for the DS 570 course during the Spring 2025-2026 semester at Özyeğin University.

The project aims to demonstrate a complete data science workflow, from raw data acquisition to the deployment of an interactive application.

Key Features & Pipeline
In accordance with the course standards, this project integrates the following core data science components:


Advanced Data Processing: Comprehensive cleaning, transformation, and feature engineering of real-world datasets.  


Exploratory Data Analysis (EDA): Insightful and interactive data visualizations.  


Machine Learning Modeling: Implementation, tuning, and rigorous evaluation of predictive models.  


Interactive Dashboard: A user-friendly interface developed using Streamlit to interact with data and model outputs.  


Version Control: A transparent development history managed via Git to showcase progress over multiple weeks.
